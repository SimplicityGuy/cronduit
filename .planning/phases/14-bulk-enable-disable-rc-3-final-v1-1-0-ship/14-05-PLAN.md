## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| browser checkbox state → POST body | untrusted; sanitized by server-side CSRF + axum_extra::Form parsing |
| inline JS → HTMX submit | browser-local; no new cross-origin concerns |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-14-05-01 | Tampering | `hx-preserve` silently fails (no stable id) → every 3s poll wipes selection | mitigate | Per-row checkbox MUST have `id="cd-row-cb-{{ job.id }}"`. Test-guarded via the Research Landmine §3 + acceptance criteria grep. |
| T-14-05-02 | Tampering | Sticky action bar placed INSIDE `<div class="overflow-x-auto">` wrapper gets clipped horizontally | mitigate | Insertion point is SIBLING, between L36 filter-bar close and L39 overflow wrapper open. Grep asserts the bar is at the top level of the content block, not descendant of the wrapper. |
| T-14-05-03 | Spoofing | CSRF token missing from bulk-toggle hx-include → POST 403s on every click | mitigate | Dashboard.html emits a hidden `<input name="csrf_token">` outside the polled `<tbody>`; `hx-include=".cd-row-checkbox:checked, [name='csrf_token']"` harvests it. |
| T-14-05-04 | Information Disclosure | Row-level override state leaks to dashboard operator through subtle visual cues without auth context | accept | v1.1 UI is unauthenticated by design; ALL operator-visible state is equally unauth; adding selection chrome does not increase exposure |
| T-14-05-05 | Tampering | XSS via job name in `aria-label="Select {{ job.name }}"` | mitigate | askama escapes `{{ }}` by default (HTML-entity encoding); attribute-context injection is safe under this rule. Manual review: no `{{ job.name | safe }}` anywhere in the new markup. |

       ```html
       <script>
       function __cdBulkOnRowChange() {
         __cdBulkUpdateBar();
         __cdBulkUpdateIndeterminate();
       }
       function __cdBulkUpdateBar() {
         const n = document.querySelectorAll('.cd-row-checkbox:checked:not(#cd-select-all)').length;
         const bar = document.getElementById('cd-bulk-action-bar');
         const countEl = document.getElementById('cd-bulk-count');
         if (!bar || !countEl) return;
         countEl.textContent = n;
         if (n === 0) bar.setAttribute('hidden', '');
         else bar.removeAttribute('hidden');
       }
       function __cdBulkUpdateIndeterminate() {
         const all = document.querySelectorAll('.cd-row-checkbox:not(#cd-select-all)');
         const checked = document.querySelectorAll('.cd-row-checkbox:checked:not(#cd-select-all)');
         const selectAll = document.getElementById('cd-select-all');
         if (!selectAll) return;
         if (checked.length === 0) {
           selectAll.indeterminate = false;
           selectAll.checked = false;
         } else if (checked.length === all.length) {
           selectAll.indeterminate = false;
           selectAll.checked = true;
         } else {
           selectAll.indeterminate = true;
           selectAll.checked = false;
         }
       }
       function __cdBulkSelectAll(on) {
         document.querySelectorAll('.cd-row-checkbox:not(#cd-select-all)').forEach(cb => {
           cb.checked = on;
         });
         __cdBulkUpdateBar();
         __cdBulkUpdateIndeterminate();
       }
       function __cdBulkClearSelection() {
         document.querySelectorAll('.cd-row-checkbox:checked').forEach(cb => { cb.checked = false; });
         __cdBulkUpdateBar();
         __cdBulkUpdateIndeterminate();
       }
       document.body.addEventListener('htmx:afterSwap', function(e) {
         if (e.detail && e.detail.target && e.detail.target.id === 'job-table-body') {
           __cdBulkUpdateBar();
           __cdBulkUpdateIndeterminate();
         }
       });
       </script>
       ```
