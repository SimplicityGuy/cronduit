**Step B ŌĆö Insert the new subsection.** Use the markdown below verbatim (adjust the heading level ŌĆö `###` or `####` ŌĆö to match the surrounding structure):

```markdown

### Tag Filter Chips

Tag your jobs in TOML; the dashboard auto-renders filter chips for every distinct
tag in your fleet. Click a chip to filter the dashboard to jobs carrying that tag.
Multiple active chips compose with **AND** semantics ŌĆö only jobs with ALL active
tags render. The active set composes with the existing v1.0 name-filter via AND
(jobs must match BOTH name LIKE filter AND every active chip).

Configure tags per job in `cronduit.toml`:

```toml
[[jobs]]
name = "nightly-postgres-backup"
schedule = "0 2 * * *"
type = "docker"
image = "postgres:16-alpine"
command = ["pg_dumpall"]
tags = ["backup", "postgres", "nightly"]
```

Tag rules (validated at config-load):
- Charset: `^[a-z0-9][a-z0-9_-]{0,30}$` (ASCII lowercase + digits + `_` + `-`; 31 char max).
- Per-job cap: 16 tags (substring-collisions across the fleet are rejected too ŌĆö `back` and `backup` cannot coexist).
- Tags are not supported in `[defaults]` (per-job only). Reserved names (`cronduit`, `system`, `internal`) are rejected.

Chip strip behavior:
- **Empty state** ŌĆö when no job has any tags, the chip strip is hidden entirely (the dashboard looks identical to v1.0/v1.1).
- **Untagged-hidden** ŌĆö when ANY chip is active, untagged jobs are hidden from the dashboard. Removing all chips restores the unfiltered view.
- **Bookmarkable URLs** ŌĆö the active set is encoded as repeated `?tag=` query params (e.g., `/?tag=backup&tag=weekly`). Copy the URL bar to share a filtered dashboard view; pasting that URL into a fresh tab loads the same chips active on first paint.
- **Stale tags** ŌĆö if you remove `tags = [...]` from a job, any bookmarked URL containing that tag silently drops the unknown tag (no error, no 500).

Tags are also delivered in webhook payloads (the `tags` field of the `run_finalized` event ŌĆö see [docs/WEBHOOKS.md](docs/WEBHOOKS.md)). They are NOT exposed as Prometheus labels (cardinality discipline; same posture as exit codes).
```
