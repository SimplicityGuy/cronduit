## Sampling Rate

- **After every task commit:** `yamllint .github/workflows/*.yml` when a workflow file is edited; `cat` + diff review for docs-only tasks; no automated Rust test change
- **After every plan wave:** full `cargo nextest run` (unchanged ‚Äî to confirm the phase did not accidentally touch Rust)
- **Before `/gsd-verify-work`:** Maintainer has run `scripts/verify-latest-retag.sh` and captured output; main-build.yml has executed at least once on `main` after landing
- **Max feedback latency:** ~15 min (bounded by a single main-build.yml CI run)
