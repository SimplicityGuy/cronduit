<objective>
Wire everything together: orphan reconciliation at startup, Docker executor dispatch in run.rs, and finalize_run with container_id for image digest recording. This is the integration plan that connects Plans 01 and 02 into the running system.

Purpose: Without this wiring, the Docker executor exists as dead code. Orphan reconciliation (SCHED-08) is the safety net that prevents container leaks across restarts -- it must run before the scheduler loop starts. The finalize_run update enables recording the image digest per DOCKER-09.

Output: `src/scheduler/docker_orphan.rs`, updated `run.rs`, `queries.rs`, `cli/run.rs`
</objective>
