| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-22-01-01 | Tampering | TOML deserialization of `tags` array | mitigate | serde rejects malformed JSON/TOML; only `Vec<String>` shape accepted; Plan 02 validators reject empty strings, invalid charset, reserved names, and collisions before any DB write |
| T-22-01-02 | Information Disclosure | tag values in DB column | accept | tags are organizational metadata explicitly chosen by the operator; not secrets; documented as plain-text in `jobs.tags` column |
